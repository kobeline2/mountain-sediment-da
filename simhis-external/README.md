# simhis-external

この領域は、外部で実行される `simhis` と疎結合に連携するための受け渡し専用領域です。

このリポジトリでは `simhis` 本体のコードやビルドは管理しません。
代わりに、以下だけを扱います。

- `specs/`
  - 受け渡すファイル形式やケース定義の仕様
- `templates/`
  - ケース記述やメタデータの雛形
- `exports/`
  - 外部実行者に渡す入力パッケージ
- `imports/`
  - 外部実行から受け取った結果やメタ情報

この分離によって、`hydrophone-gp/` や `thesis/` は
`simhis` の実行環境に依存せずに運用できます。
